GitHub Sync

Wie funktioniert der Sync?

Die App speichert alle Buchungsdaten in einem privaten GitHub Gist. Beide Geräte (PC + Smartphone) laden und speichern daraus – so bleibt alles synchron, ohne eigenen Server.

Du brauchst: einen GitHub Account + ein Personal Access Token mit gist-Berechtigung.
Token erstellen: GitHub → Settings → Developer Settings → Personal Access Tokens → Fine-grained → Gists: Read & Write.
